# salt主题
基于notion主题修改的护眼配色主题，增添一番风味。
![image.png](https://tva1.sinaimg.cn/large/8ff342a8gy1h2wjjwfqm7j20u01d9wjq.jpg)
---
### 显示顶栏方式：
* 把主题文件里的`n-topbar.css`文件拷贝到工作空间`data\widgets`目录里面
* 此方法适用于：notion-theme、notion-dark和salt主题
